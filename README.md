# spring-jwt-rest-api
Simple rest API with spring boot with JWT token Authentication.



## Setup
- in the src/main/resources/application.properties change the mogo uri to your URI. <br>
`spring.data.mongodb.uri=mongodb://your_host/your_db`

- MongoDb
